# homebrew-tap

Homebrew formulae for [artisenalcode](https://github.com/artisenalcode) tools.

```bash
brew install artisenalcode/tap/crunchit
```

| Formula | Description |
|---|---|
| [crunchit](https://github.com/artisenalcode/crunchit) | Fast CLI for optimizing images and generating WebP/AVIF variants |

Formulae install prebuilt release binaries (checksums pinned per release).
